# SIGCEC - Sistema Integral de Gestión de Citas y Expedientes Clínicos

## Descripción
SIGCEC es un sistema web que automatiza la gestión de citas médicas y el manejo seguro de expedientes clínicos. El objetivo es eliminar procesos manuales, mejorar la comunicación con los pacientes y garantizar la seguridad de la información.
